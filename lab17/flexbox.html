Challenge

This labs challenge was to use advanced css to do fancy stuff and things.

Problems

This lab was much easier for me compared to the last one. I was already using box shadow, transform, text-shadow and rounded corner in other labs so that was a good refresher. The hardest part for me was the filters. They confused me because some were precentages, some decimals, and some degrees.

Results

I was able to create the file directory correctly, create divs, link my css file, edit margins and padding, position the photos, applied text-shadow, box-shadow, gradient, filters, rounded corners, object fit and transformed photos.
